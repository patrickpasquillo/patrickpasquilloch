Inserisci le registrazioni contabili immobiliari

Immobili in affitto

Per inserire le registrazioni contabili bisogna utilizzare anche i campi CC1, CC2 e CC3 per gli immobili in affitto e almeno CC1 per i condomini.

Nel campo CC1, cliccandoci sopra, bisognerà inserire una tra queste voci o lasciare vuoto:

  • AFFITTO quando si incassa affitto e/o acconto spese dell'inquilino
  • ALTRI quando si incassa altro che non sia affitto e acconto spese tranne per i conguagli (per esempio spese amministrative)
  • CONGUAGLIO quando si incassa un conguaglio spese
  • CONTEGGIO quando si vuole girare le spese nel conto contabile di conteggio per farle ripartire dall’estensione immobiliare.
  • Lasciare vuoto quando si tratta di una registrazione contabile di costo o di ricavo non imputabile a queste situazioni.

La voce ACCONTI viene utilizzata solo durante l’addebito automatico dell’affitto mensile attraverso le voci dell’estensione.

Nel campo CC2 si consiglia di inserire l’immobile relativo alla registrazione contabile, soprattutto se si gestiscono più immobili nello stesso file di contabilità.

Nel campo CC3 si consiglia di inserire per ogni costo l’immobile (costo per i vani comuni) o l’appartamento (costo di manutenzione relativo all'appartamento indicato). Questo campo sarà utile per avere la lista delle manutenzioni svolte all’interno dei singoli appartamenti o dei vani comuni. La lista delle manutenzioni stampa tutte le manutenzioni eseguite da quando vengono gestite le contabilità con l'estensione immobiliare.

Condomini

Nei condomini troviamo invece le voci ACCONTO, ALTRO, CONGUAGLIO E FR.

È necessario inserire:

  • ACCONTO quando si incassa un acconto spese del comproprietario
  • CONGUGAGLIO quando si incassa un conguaglio spese
  • ALTRO quando si incassa altro che non sia Acconto spese, conguaglio o fondo di rinnovamento (per esempio spese per telecomandi)
  • FR quando si incassa l’importo di un Fondo di rinnovamento
  • Lasciare vuoto quando si tratta di una registrazione contabile di costo o di ricavo non imputabile a queste situazioni.

Il campo CC2 è nascosto per i template relativi ai Condomini perché l’immobile è sempre uno solo.

Il campo CC3 è nascosto ma si può inserire per ogni costo l’immobile o l’appartamento relativo alla spesa sostenuta. Questo campo sarà utile per avere la lista delle manutenzioni svolte all’interno di ogni singolo appartamento o nei vani comuni. Chiaramente le spese sostenute all'interno di un appartamento solitamente sono a carico del comproprietario ma in caso di sinistri assicurativi comuni possono essere utilizzate. La lista delle manutenzioni stampa tutte le manutenzioni eseguite da quando vengono gestite le contabilità con l'estensione immobiliare.
